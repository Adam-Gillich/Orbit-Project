# AI usage statement

This project used ChatGPT for:
- checking algebraic derivations
- converting equations into LaTeX
- explaining orbital mechanics concepts
- verifying consistency of numerical methods
- numerical implementation tips
- obtaining sources

All physics assumptions, parameter choices, and final conclusions
were made by the author.

This project used multiple avalible models of GitHub for coding:
+ general fixes
+ LaTex debugging
+ mermaid diagramms

Relevant conversations are documented in this file.
